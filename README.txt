caesar.c
--------
-> Accepts a numerical key as command line argument
-> Enter a plain text to be encoded
-> The key is used to encode the plain text using caesar cipher
-> Encoded text as output
For more on caesar cipher: https://en.wikipedia.org/wiki/Caesar_cipher

vigenere.c
----------
-> Accepts a keyword as command line argument
-> Enter a plain text to be encoded
-> The keyword is used to encode the plain text using vigenere cipher
-> Encoded text as output
-> For more on vigenere cipher: https://en.wikipedia.org/wiki/Vigen%C3%A8re_cipher
